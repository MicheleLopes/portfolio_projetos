Projetos em Ciência de Dados

Desenvolvimento de um ETL e Dashboard para monitoramento do aprendizado dos alunos na Comunidade DS

Este projeto foi desenvolvido através de um vínculo por uma empresa Jr. da Comunidade DS e em colaboração com o cientista de dados Daniel Salvatori. Ele possui duas entregas principais:

  • ETL : Responsável pela extração dos dados da plataforma de ensino, manipulação e carga em tabelas dentro do GCP (Google Cloud Platform);
  • Dashboard : Painel para interação com o usuário utilizando a plataforma Streamlit, onde será possivel o acompanhamento do progresso dos alunos.;

Resultados:

  • Redução no tempo de coleta de dados de 2-4 horas para menos de 1 minuto;
  • Facilitação na análise do desempenho individual e por turma, permitindo intervenções mais rápidas e assertivas;
  • Melhoria na qualidade dos dados, eliminando o risco de erro humano;
  • Possibilidade de gerar relatórios para análises estratégicas e operacionais.

As ferramentas utilizadas foram:

  • Python, Pandas, Plotly e Numpy;
  • VSCode;
  • Git e github;
  • Streamlit;
  • Google Cloud Platform (GCP);
  • Integração de API.

Predição de vendas para lojas de uma rede farmacêutica

Desenvolvimento de um projeto de Machine Learning para solucionar um problema de regressão de predição de vendas de 1115 lojas nas próximas 6 semanas, para auxiliar os gerentes de loja na tomada de decisão de futuros investimentos.

O produto final foi um bot dentro do aplicativo Telegram, que pode ser acessado pelo celular e através de uma API, retorna as previsões.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy, matplotlib e Seaborn;
  • Anaconda e Jupyter Lab;
  • Algoritmo Regressão, Machine Learning e Sklearn;
  • Técnicas de Feature Selection e Redução de Dimensionalidade;
  • Render Cloud;
  • Framework Python Flask;
  • API Telegram para criação de Bots;
  • Git e Github.
